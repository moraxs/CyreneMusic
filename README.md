# Cyrene Music 🎵

一个功能完善的跨平台音乐播放器，使用 Flutter 开发。


## 📱 支持平台

- ✅ Windows
- ✅ Android  

## 🚀 快速开始

### 前端运行

```bash
# 安装依赖
flutter pub get

# 运行应用
flutter run

# 构建 Windows 版本
flutter build windows

# 构建 Android APK
flutter build apk
```

### 后端运行

```bash
cd backend

# 安装依赖
bun install

# 启动服务器
bun run src/index.ts
```
